# PDF Editor 開発履歴

---

## v1.0.0 — 初回リリース

### 追加機能
- PDF ファイルの読み込み・保存（上書き／名前を付けて保存）
- ページサムネイル一覧表示
- プレビュー表示（拡大・縮小・スクロール）
- ページ回転（90° / 180° / 270°）— 複数ページ対応
- ページ削除 — 複数ページ対応
- トリミング（上下左右の余白を数値入力で削減）
- 別 PDF からページ挿入（単一ファイル）
- PDF 結合（単一ファイル、末尾に追加）
- ページ移動（番号指定）
- Ctrl+クリックによる複数ページ選択
- 全選択 / 選択解除ボタン
- 起動用バッチファイル（ライブラリ自動インストール付き）

---

## v1.1.0 — 結合機能強化

### 追加・変更機能
- **複数ファイル結合対応**: 結合・挿入で複数 PDF を同時選択可能
- **結合順ダイアログ新設**: 選択ファイルの順番を GUI で並び替え可能
  - ▲▼ ボタンによる順番変更
  - リストからの個別除外
  - 結合後の合計ページ数をリアルタイム表示
- 挿入機能も複数ファイル対応（選択順に指定位置へ順次挿入）

---

## v1.2.0 — トリミング UI 刷新

### 追加・変更機能
- **ドラッグ選択トリミング**: プレビュー上でマウスドラッグして残す範囲を直接指定
  - 選択範囲外を暗幕で表示
  - 選択中の座標・サイズ（pt）をリアルタイム表示
  - 範囲リセットボタン追加
- 数値入力によるトリミング UI を廃止

---

## v1.3.0 — ページ並び替え強化・終了ボタン追加

### 追加・変更機能
- **サムネイルのドラッグ＆ドロップによるページ入れ替え**
  - ドラッグ中にゴースト（半透明プレビュー）を表示
  - 挿入位置を赤い点線インジケーターで表示
  - ドラッグしない場合は従来通りクリック選択として動作
- **終了ボタン追加**（ファイルを開いている場合は確認ダイアログあり）
- 起動バッチをシンプル化（`cd /d "%~dp0"` で同フォルダ起動に統一）

---

## v1.4.0 — パフォーマンス改善・安全性強化・Undo対応

### バグ修正
- **WM_DELETE_WINDOW 未設定**: ウィンドウ「×」ボタンで閉じた場合も確認ダイアログを表示し `doc.close()` を実行するように修正
- **上書き保存の二重例外処理漏れ**: incremental 保存失敗時のフォールバック保存が更に失敗した場合のエラーハンドリングを追加
- **全ページ削除防止**: 全ページを選択して削除しようとした場合に警告を表示し、最低1ページを維持するよう制御
- **D&D バウンダリチェック強化**: サムネイルパネル外へドロップした場合の挿入先計算を安全に処理

### パフォーマンス改善
- **サムネイルキャッシュ化**: `thumb_cache` 辞書を導入し、ページ変更がないサムネイルの再レンダリングを回避。大量ページPDFでの操作速度が大幅向上
- **選択変更時のプレビュー再描画抑止**: 全選択・選択解除・Ctrl+クリック時はサムネイル背景色のみ更新し、プレビュー再描画をスキップ（`_refresh_thumbs_selection_only` 新設）
- **トリミング暗幕の描画最適化**: ドラッグ中に暗幕を毎フレーム削除→再生成していたのを `canvas.coords()` による座標更新方式に変更
- **MergeOrderDialog のページ数キャッシュ**: 結合順ダイアログ内の並び替え操作で毎回PDFファイルを開いてページ数を数えていたのを、初回のみ取得してキャッシュに変更

### 追加機能
- **Undo / Redo（Ctrl+Z / Ctrl+Y）**: 回転・削除・トリミング・挿入・結合・ページ移動のすべての操作を最大20回まで取り消し・やり直し可能（ツールパネルにボタン追加）
- **キーボードショートカット**: Ctrl+O（開く）、Ctrl+S（保存）、Ctrl+Shift+S（名前を付けて保存）、Ctrl+Z / Ctrl+Y（Undo/Redo）、Delete（ページ削除）
- **ファイル D&D 対応**: エクスプローラーから PDF をウィンドウにドロップして開く機能（`windnd` ライブラリ使用、未インストール時はサイレントスキップ）

### UI 改善
- **上書き保存に確認ダイアログ追加**: 誤操作による上書きを防止
- **トリミングモードの視覚強調**: ON 時にボタンが赤背景＋🔴アイコンに変化し、モード状態が一目で分かるように改善
- **挿入位置ダイアログの説明文改善**: 「例: 3 と入力 → 2ページ目と3ページ目の間に挿入」のように具体例を表示
- **結合ダイアログにページ数表示**: 各ファイル名の横にページ数を表示（例: `report.pdf (12p)`）
- **ボタンにショートカットキー表記追加**: 主要ボタンにキーバインド情報を併記

### コード品質
- **未使用引数の削除**: トリミング座標変換関数の不要な `off` パラメータを除去
- **終了時の doc.close() 確実化**: `_quit` メソッドで必ず `doc.close()` を呼んでからウィンドウを破棄
