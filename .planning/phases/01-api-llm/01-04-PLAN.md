## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| _on_run / _on_summary → _confirm_cost / _confirm_summary_cost → ユーザー | クラウド送信の直前に「送信先ホスト・課金」をユーザーへ開示し承認を得る境界。ここで開示される送信先が実際の送信先と一致していなければならない |
| _confirm_cost / _confirm_summary_cost → app.settings.get("runpod_url" / "runpod_model") | 送信先/モデルの真の源。読み取りのみ（本プランで書き込みは追加しない） |
| ページ画像 / OCR テキスト → ユーザー設定の runpod_url（外部・課金付き HTTPS 送信） | 承認後に機微データが越境する境界。承認前の開示内容がこの実送信先を正しく反映している必要がある |
