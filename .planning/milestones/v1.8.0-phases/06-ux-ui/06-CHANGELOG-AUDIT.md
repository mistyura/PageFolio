### 検出した不一致の詳細（v1.6.1）

**症状:** `開発履歴.md` の v1.6.1 見出し・バージョン索引テーブル・最終更新サマリの 3 箇所すべてで日付が **2026-06-22** と記載されていたが、実際には:

- `git tag v1.6.1` はコミット `fd20608eaff8619a671f591feb06fe971685585e`（PR #25 マージコミット）を指しており、そのコミット日は **2026-06-23**
- `.planning/MILESTONES.md` の該当エントリも `Shipped: 2026-06-23` と明記
- v1.6.1 のセクション本文が記載する2機能のうち、OCR タイムアウト拡大（`2bff34b`, 2026-06-22）は 2026-06-22 に実装されたが、**パスワード対応・印刷機能**（STATE.md Quick Tasks 記録の `260623-pwp`）は翌日 **2026-06-23** に実装され、同日の `260623-rel` で PR #25 マージ・タグ付け・GitHub Release 公開まで完了している

つまり v1.6.1 は「2026-06-22 に着手し 2026-06-23 に完成・出荷」した版であり、開発履歴.md がスタンプすべきは他の全エントリと同じ基準（タグ作成日＝ショップ日）に倣うなら **2026-06-23** が正しい。2026-06-22 という記載は、2機能のうち先に実装されたタイムアウト拡大のコミット日のみを反映した誤りだったと判定した。

**修正内容:**
