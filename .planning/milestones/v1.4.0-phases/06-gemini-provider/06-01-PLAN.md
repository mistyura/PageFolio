## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| アプリ → Gemini API（generativelanguage.googleapis.com） | API キーと PDF ページ画像（base64）が外部へ送信される境界 |
| Gemini API → アプリ（レスポンス JSON） | 信頼できない外部入力（candidates 構造・SAFETY ブロック） |
| 環境変数 / セッションメモリ → Provider | API キーの読み取り経路（書き込み禁止） |
