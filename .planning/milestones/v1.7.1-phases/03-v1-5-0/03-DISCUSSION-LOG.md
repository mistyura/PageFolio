## 画像透かしの仕様

### UI 導線

| Option | Description | Selected |
|--------|-------------|----------|
| ファイル選択→即適用 | テキスト透かしと同じミニマムパターン。ボタン→filedialog→既定値で即適用 | ✓ |
| 設定付きダイアログ | ファイル選択後に配置・サイズ・不透明度を指定して適用 | |
| プレビュー付きダイアログ | 適用前にサムネイルプレビューで仕上がり確認 | |

### 既定配置・サイズ

| Option | Description | Selected |
|--------|-------------|----------|

| タイル敷き詰め | 小さめロゴをページ全面に繰り返し配置 | |
| 四隅に小さく | ページ隅にロゴ配置（ブランド刻印用途） | |

### 不透明度・回転

| Option | Description | Selected |
|--------|-------------|----------|
| 50%透過・回転なし | テキスト透かしの透過と揃えつつロゴは水平。透過PNGのアルファは乗算尊重 | ✓ |
| 50%透過・45°斜め | テキスト透かしと完全に同じ見た目ルール | |
| 原画のまま | アプリ側は無加工で埋め込み | |

### 対応画像形式

| Option | Description | Selected |
|--------|-------------|----------|
